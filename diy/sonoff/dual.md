# DUAL

## Sonoff Dual R1

| Pin      | Function                |
| -------- | ----------------------- |
| GPIO1  | **Relay #1**            |
| GPIO3  | **Relay #2**            |
| GPIO13 | **Blue LED (inverted)** |
| GPIO4  | Optional sensor         |
| GPIO14 | Optional sensor         |

## Sonoff Dual R2

| Pin      | Function                |
| -------- | ----------------------- |
| GPIO12 | **Relay #1**            |
| GPIO5  | **Relay #2**            |
| GPIO10 | **Button**              |
| GPIO13 | **Blue LED (inverted)** |
| GPIO4  | Optional sensor         |
| GPIO14 | Optional sensor         |


## 使用说明
BUTTON0 = GPIO0，所以连接 BUTTON0 + GND 进入刷机模式
